# 波兰式

  • 数学计算中将运算符放在运算数字之前形成前缀表达式
  • 从左至右扫描表达式，如果一个操作符后面跟着两个操作数时，则计算，然后将结果作为操作数替换(这个操作符和两个操作数)，重复此步骤，直至所有操作符处理完毕
